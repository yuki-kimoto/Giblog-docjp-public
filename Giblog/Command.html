Giblog - Gitで管理できるWebサイトとブログの作成ツール

名前

Giblog::Command - コマンド基底クラス

説明

Giblog::Commandは、コマンドの基底クラスです。

Giblog::Commandを継承して、Giblog::Command::new、Giblog::Command::add、Giblog::Command::buildのような、オリジナルのコマンドを作成できます。

メソッド

new

$command->new(%args);

Giblog::Commandオブジェクトを生成します。

引数:

  • api

    Giblog::APIオブジェクト

api

$command->api;

Giblog::APIオブジェクトを取得します。

run

$command->run(@args);

コマンドを実行します。このメソッドは、サブクラスで実装されます。
