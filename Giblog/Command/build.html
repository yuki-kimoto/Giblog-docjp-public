Giblog - Gitで管理できるWebサイトとブログの作成ツール

名前

Giblog::Command::build - Webサイト構築コマンド

説明

Giblog::Command::buildはWebサイトを構築するためのコマンドです。

メソッド

Giblog::Command::buildはGiblog::Commandのすべてのメソッドを継承しており、次の新しいメソッドを実装しています。

run

$command->run;

Webサイトを構築します。

このメソッドは、サブクラスでオーバーライドされることが予定されています。
