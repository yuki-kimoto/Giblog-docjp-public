Giblog - Gitで管理できるWebサイトとブログの作成ツール

名前

Giblog::Command::new - 空のWebサイト作成コマンド

説明

Giblog::Command::newは、空のWebサイトを作成するためのコマンドです。

Giblog::Command::newを継承して、Giblog::Command::new_blogやGiblog::Command::new_websiteのような、オリジナルのWebサイト作成コマンドを作ることもできます。

メソッド

Giblog::Command::newはGiblog::Commandからすべてのメソッドを継承しており、次の新しいメソッドを実装しています。

run

$command->run($website_name);

Webサイト名を指定して、Webサイトを作成します。

すべてのコンテンツは、protoディレクトリからコピーされます。

もしモジュールがロードされたパスが"/path/Giblog/Command/new.pm"の場合は、protoディレクトリのパスは"/path/Giblog/Command/new/proto"になります。
