Giblog - Gitで管理できるWebサイトとブログの作成ツール

名前

Giblog::Command::serve - Webサイトのサーブコマンド

説明

Giblog::Command::serveはWebサイトをサーブするコマンドです。

メソッド

Giblog::Command::addはGiblog::Commandからすべてのメソッドを継承しており、次の新しいメソッドを実装しています。

run

$command->run;

Mojoliciousのmorboコマンドを使って、serve.plでWebサイトサーブします。

以下のコマンドと同じです。

morbo -w giblog.conf -w lib -w templates serve.pl
