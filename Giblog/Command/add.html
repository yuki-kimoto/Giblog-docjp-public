Giblog - Gitで管理できるWebサイトとブログの作成ツール

名前

Giblog::Command::add - 新しいブログエントリーを追加するコマンド

説明

Giblog::Command::addは、新しいブログエントリーを追加するためのコマンドです。

メソッド

Giblog::Command::addはGiblog::Commandからすべてのメソッドを継承しており、次の新しいメソッドを実装しています。

run

$command->run;

「templates/blog」ディレクトリの中に新しいブログエントリーページのファイルを作成します。

ファイルは、日付と時刻を含みます。

templates/blog/20190416153053.html
